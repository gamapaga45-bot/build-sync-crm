# BuildSync CRM — Интеллектуальный портал управления строительством

BuildSync CRM — это современное Full-stack приложение для управления строительными объектами, ресурсами и клиентами. Система объединяет управление задачами, финансовый учет, BIM-визуализацию, ИИ-ассистента и систему облачной фотофиксации в единый интерфейс.

## 🌟 Основные возможности

- **Командный центр**: Единая панель управления всеми модулями системы.
- **Облачная фотофиксация**: Интеграция с Cloudflare R2 для надежного хранения фотографий объектов.
- **Live-трансляции**: Система прямых эфиров со стройплощадки для технадзора и инженеров.
- **Управление задачами**: Канбан-доска, фотоотчеты и отслеживание прогресса в реальном времени.
- **Финансы и Биллинг**: Учет смет, ведение счетов, анализ маржинальности проектов.
- **BIM & Карты**: Просмотр 3D-моделей зданий и геопривязка объектов.
- **AI-Ассистент**: Интеллектуальные подсказки по нормативам и автоматизация рабочих процессов.

---

## 🛠 Технологический стек

- **Frontend**: React 19, TypeScript, Tailwind CSS v4, Framer Motion.
- **Backend**: Node.js, Express, Socket.io (для Real-time).
- **Storage**: Cloudflare R2 (S3-compatible).
- **Auth & DB**: Firebase (Authentication, Firestore).
- **AI**: Google Gemini.

---

## 🚀 Предварительные требования

Для полноценной работы системы вам потребуются:
1.  **Google Gemini API Key**: Для работы ИИ-ассистента.
2.  **Firebase Project**: Для авторизации пользователей и хранения данных.
3.  **Cloudflare Account**: Для хранения фотографий (R2) и потенциально базы данных (D1).

## ⚙️ Локальная разработка

1.  **Установите зависимости**:
    ```bash
    npm install
    ```
2.  **Настройте окружение**:
    Создайте файл `.env` на основе `.env.example` и заполните ключи:
    - `GEMINI_API_KEY`
    - `CLOUDFLARE_ACCOUNT_ID`, `R2_ACCESS_KEY_ID`, `R2_SECRET_ACCESS_KEY`, `R2_BUCKET_NAME`
    - Перетащите конфиг Firebase в `firebase-applet-config.json`

3.  **Запустите сервер разработки**:
    ```bash
    npm run dev
    ```
